# IPL Performance Dashboard 📊

## Overview

This project presents an interactive Power BI dashboard analyzing IPL data to uncover insights about players and teams.

## Features

* Interactive team-level filtering using slicer
* Identification of top run scorers and wicket takers
* Analysis of runs across match phases (Powerplay, Middle, Death)
* Clean and intuitive dashboard design

## Tools Used

* Power BI
* Python (for data cleaning)
* Data Visualization

## Key Insights

* Top players contribute significantly to team performance
* Middle overs generate the highest runs
* Team performance varies significantly across matches
